Vulture Decompiler

Python

Comprehensive toolkit for analyzing, decompiling, and deobfuscating Minecraft mods. Features static analysis, multiple decompilers (CFR, JD-CLI, Fernflower), automatic tool installation, and security detection. Impact: Safe isolated execution in Docker containers. Automatically detects Minecraft versions and downloads mappings. Identifies suspicious patterns like webhooks and token access.

Architecture: JAR Input → Version Detection → Decompiler Selection → MCP Mapping Download → Deobfuscation → Source Output. Docker-based isolation for security. Supports recompilation back to JAR files.

Sora Prompt Generator Bot

Python

Lightweight Discord bot that generates viral Sora 2 video prompts using Groq API. Docker-ready with resource limits, 10 prompt templates, and fast responses. Designed for mini PC deployment. Impact: Automated prompt generation with multiple styles (CCTV, body cam, Olympic broadcast, etc.). Resource-efficient design uses <256MB memory.

Architecture: Discord Command → Groq API → Template Engine → Response. Docker containerized with CPU/memory limits. Automatic reconnection and error handling.

Discord AI Chatbot

Python

Production-ready Python chatbot with fully programmable personality system. Responds in chat channels with Groq API integration. Automatic rate limit handling, token-aware throttling, and optimized conversation history. Impact: Handles 30 requests/minute with automatic throttling.
