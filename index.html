ECS Service Deployment

AWS

Built a production-ready containerized application deployment system on AWS ECS (Fargate) with Application Load Balancer routing. The project solves the problem of manual deployments and infrastructure drift by automating the entire CI/CD pipeline through GitHub Actions.

What I learned: Deep dive into AWS networking (VPCs, security groups, ALB target groups), container orchestration best practices, and implementing zero-downtime blue-green deployments. Used CloudFormation for infrastructure-as-code.

GPU 3D Asset Pipeline

Python

Developed an RTX 4090-powered workflow that automates mesh and texture generation for Roblox assets using AI-assisted modeling. The system processes hundreds of assets in parallel, reducing manual 3D modeling time from hours to minutes per asset.

Tech highlights: CUDA acceleration, batch processing with multiprocessing, AI model integration for texture generation, and automated UV mapping. Built custom Python tools for mesh optimization and format conversion.

AI-Powered Discord Bot

Node.js

Created an intelligent Discord chatbot using AI APIs, built with agentic development workflows in Cursor. The bot maintains conversation context, handles multiple simultaneous users, and is fully containerized for easy deployment and scaling.

Key features: Context-aware responses, rate limiting, error handling, Docker containerization, and integration with Discord's slash commands. Designed for 24/7 uptime with automatic reconnection and health monitoring.
